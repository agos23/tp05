10-¿Qué son los métodos / verbos de petición HTTP? Dar ejemplos. Bibliografía MDN

Los metodos o vervos HTTP indican la accion que desea realizar un recurso determinado ,cada uno de ellos tiene caracteristicas similares Metodos HTTP:
GET:solo deven recuperar datos,solicita una reprecentacion de un recurso especifico.
HEAD:pide una respuest identica a la deuna peticion GET.
POST:replaza las reprecentaciones actuales del recurso.
DELETE:borra un recurso en especifico.
CONNECT:crea un tunel hacia el servidor identificado para el recurso.
OPTIONS: se usa para escribir opciones para el recurso de destino.
TRACE:hace una prueva de mensaje a lo largo de la ruta al recurso de destino.
PATCH:se usa para hacer cambios parciales a un recurso.
